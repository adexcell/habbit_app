# Habbit App 📝

**Habbit App** — это интуитивно понятное веб-приложение для отслеживания ваших ежедневных привычек и мониторинга прогресса. Создавайте новые цели, ставьте перед собой задачи и следите за тем, как вы становитесь лучше с каждым днем!

![Habbit App Demo](./images/logo.svg)

## ✨ Особенности

- **Управление привычками**: Создавайте неограниченное количество привычек.
- **Персонализация**: Выбирайте иконки для различных категорий (Спорт, Еда, Напитки).
- **Отслеживание прогресса**: Устанавливайте целевое количество дней и отмечайте выполнение каждого дня с комментариями.
- **Визуализация**: Интерактивная шкала прогресса, показывающая процент выполнения цели.
- **Удаление**: Возможность удалять как отдельные записи за день, так и привычки целиком.
- **Сохранение данных**: Все ваши данные сохраняются локально в браузере (`localStorage`), поэтому они не пропадут после перезагрузки страницы.

## 🛠 Технологический стек

- **HTML5**: Семантическая разметка.
- **CSS3**: Современная стилизация с использованием Flexbox и CSS-переменных.
- **Vanilla JavaScript**: Логика приложения без использования внешних библиотек и фреймворков.
- **Google Fonts**: Шрифт "Comfortaa" для приятного пользовательского интерфейса.

## 📁 Структура проекта

- `index.html` — главная страница приложения.
- `styles/` — CSS-файлы (основной стиль в `main.css`).
- `scripts/` — JavaScript-логика (основной файл `app.js`).
- `images/` — SVG-иконки и логотипы.
- `data/` — демо-данные в формате JSON.

## 🚀 Как запустить

Приложение не требует сложной настройки или сервера.

1. Склонируйте репозиторий или скачайте файлы.
2. Откройте файл `index.html` в любом современном веб-браузере.
3. Начните добавлять свои первые привычки!
